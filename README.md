 ## 铸龙-用户行为分析系统
-----------
> 铸龙-用户行为分析系统 是一款用于分析用户行为事件的BI软件，功能如下
 * 事件分析
 * 漏斗分析
 * 留存分析
 * 智能路径分析
 * 报表，面板管理

> 技术栈主要用
 * 前端：vue+elementui+antdv
 * 后端：go
 
> 所需中间件
 * mysql
 * redis
 * kafka
 * clickhouse

欢迎大家提出自己的issue。
