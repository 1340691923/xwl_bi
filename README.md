铸龙-用户行为分析系统
-----------
> 铸龙-用户行为分析系统 是一款用于分析用户行为事件的BI软件，功能如下
 * 事件分析
 * 漏斗分析
 * 留存分析
 * 智能路径分析
 * 报表，面板管理
